¿Qué necesito?

Para comenzar a usar PHP, puede:

    Encuentre un proveedor de alojamiento web con soporte para PHP y MySQL
    Instale un servidor web en su propia PC y luego instale PHP y MySQL

Utilice un servidor web con soporte PHP

Si su servidor ha activado el soporte para PHP, no necesita hacer nada.

Simplemente cree algunos archivos .php, colóquelos en su directorio web y el servidor los analizará automáticamente.

No necesita compilar nada ni instalar herramientas adicionales.

Debido a que PHP es gratuito, la mayoría de los servidores web ofrecen soporte PHP.
Configure PHP en su propia PC

Sin embargo, si su servidor no es compatible con PHP, debe:

    instalar un servidor web
    instalar PHP
    instalar una base de datos, como MySQL

El sitio web oficial de PHP (PHP.net) tiene instrucciones de instalación para PHP: http://php.net/manual/en/install.php
