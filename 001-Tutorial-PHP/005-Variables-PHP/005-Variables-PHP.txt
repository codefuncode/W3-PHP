Variables PHP

Una variable puede tener un nombre corto (como x e y) o un nombre más descriptivo (edad, nombre del automóvil, volumen_total).

Reglas para variables PHP:

Una variable comienza con el signo $, seguido del nombre de la variable
Un nombre de variable debe comenzar con una letra o el carácter de subrayado
Un nombre de variable no puede comenzar con un número
Un nombre de variable solo puede contener caracteres alfanuméricos y guiones bajos (A-z, 0-9 y _)
Los nombres de las variables distinguen entre mayúsculas y minúsculas ($ age y $ AGE son dos variables diferentes)

¡Recuerde que los nombres de variables PHP distinguen entre mayúsculas y minúsculas!
